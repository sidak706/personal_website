My Projects

As a programmer, I am always looking to work on different projects. These projects help me improve my skills and learn more about the language. I often use Python or Java to make these projects.

Jarvis: Virtual Assistant

A virutal assistant made using Python. It can perform a variety of tasks ranging from checking time and date to generating QR codes and playing music. With more than 700 lines of code, this is one of the biggest projects I have worked on.

Sudoku Solver

A sudoku solver made using Python and backtracking. The solver generates a random keyboard each time and solves the board regardless of the complexity. The user can see the backtracking algorithm in play as it solves the board.

Music Player

A GUI Music Player made using Python. The user can open the songs folder and select any song they wish to play. In addition, the user can pause and resume the song as well. This is a small project with less than 150 lines of codes.

Cancer Detection

A Cancer detecting algorithm made using Python and Machine learning. This is one of the earliest Data Science projects I worked on. The algorithm uses a simple Keras neural network and can detect whether the Cancer is malignant or benign with 80% accuracy.

Hangman Game

A Hangman GUI made using Python and pygame. The user can choose from a range of letters and depending on the answer, the hangman updates. This is one of the first games I made using Python.

Arduino Spider

A quadruped spider made using Arduino. This is one of the most advanced robotics projects I have worked on. The robot's build is very stable and it has the ability to walk around and perform different motions such as greeting, dancing etc.
